### Практическая работа. Работа со встроенными объектами и коллекциями
Создать сервис `ML` для обработки данных в виде статического объекта.

1. Создать класс `Sample`, позволяющий создать выборку числовых данных в виде массива во внутреннем приватном поле `data`. Конструктор класса должен принимать выборочные данные как в виде массива, так и в виде отдельных значений. Сохранять дату и время создания экземпляра класса в публичном свойстве `created`.

2. Предусмотреть в класе `Sample` свойства только для чтения, возвращающие основные характеристики выборочной совокупности, хранящейся в приватном свойстве `data`:

   - объём (`volume`);
   - минимальное (`minValue`) и максимальное (`maxValue`) значения;
   - размах вариации (`range`);
   - мода (`mode`);
   - медиана (`median`);
   - выборочное среднее (`mean`);
   - выборочная дисперсия (`variance`);
   - выборочное стандартное отклонение (`stDev`);
   - относительное стандартное отклонение (`cV`).

3. Предусмотреть в классе `Sample` следующие публичные методы:

    - `getData` — возвращает исходный массив данных выборки;
    - `getSeries` — возвращает исходный массив данных выборки, отсортированный по возрастанию;
    - `getValues` — возвращает набор данных выборки без дубликатов в виде множества уникальных значений, отсортированных по возрастанию;
    - `getVariationSeries` — возвращает вариационный ряд в виде ассоциативного массива, в котором ключами являются варианты (уникальные значения данных выборки), а значениями — абсолютные частоты вариант (частости — количество раз, которое конкретная варианта встречается в выборке).

4. Предусмотреть в классе `Sample` публичные свойства только для чтения, возвращающие логические значения и позволяющие оценить выборочную совокупность с точки зрения обобщенных характеристик:

   - `isNormal` — нормальность (по правилу трёх сигм);
   - `isSymmetric` — симметричность.

5. Предусмотреть в классе `Sample` публичное свойство `uniformity` только для чтения, показывающее однородность признаков выборки и возвращающее одно из четырех предустановленных значений (`"low"` | `"medium"` | `"heigh"` | `"extreme"`).
